validação do modelo com bean validation
- bean validation é uma especificação que faz parte do JEE
- objetivo é facilitar a validação de propriedades em objetos java
- adicionar anotações do bean validation que representam as validações nas classes de modelo para definir as restrições
- fornece varias anotações
- as implementações do bean validation podem fornecer validações próprias
- também podemos extender para validar coisas específicas
- precisamos de uma implementação - hibernate validator (starter web)
